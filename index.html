Projects

Praktikum 1 - Canvas HTML

This section discusses the basis of using HTML canvas as a medium for creating various kinds of web-based multimedia elements. Canvas HMTL used to draw graphics / bitmaps interactively using scripts (javascript).

Praktikum 2 - Interactivity

This section learns about interactivity in web applications using pointers. It is expected that students can understand the event using a mouse and keyboard as well take input values ​​from both devices to be developed into an application simple drawing.

Praktikum 3 - Vector Pattern

Web-based multimedia can be used for various purposes. One of them is to create digital art content. This section trains students to create content digital art uses vector canvas that is programmatically created using the concept looping in javascript.

Praktikum 4 - Puzzle Project

One interactivity using event pointer that can be done by user is do drag & drop. This activity can be applied to a variety of needs for example editing objects in a graphic editor or arranging objects as in puzzle game.

Praktikum 5 - Motion Tween

Animations can be made using Canvas HTML by changing object attributes in time intervals. On the HTML canvas, time intervals can be created using the time function interval or animation frame.

Praktikum 7 - Audio Visualizer

Presentation and capture of audio through the application the web can be developed using an HTML canvas. Through HTML canvas, audio data can be obtained then is used as an input variable to manipulate the graphic elements animated (animated graphic element).

Praktikum 8 - Online Audio Recording

Recording audio is taking audio data through a microphone device. Audio data obtained can then be used further for example: stored on the device or uploaded to the server. Using the recording capabilities on the HTML canvas one can used to make audio conversations online.

Praktikum 9 - Image Manipulating

Using the drawImage () function on an HTML canvas, image objects can be manipulated to size and how it looks. This function can be used to create image editing tools, especially for cropping purposes.

Praktikum 10 - Online Photo Capturing

One of the devices that can be accessed through a web application is the camera. Camera it can be used to take one frame as a photo. Photo data is in the form of DATA_URL with base64 format. The data can then be utilized to be sent to the server or downloaded in PNG format.
